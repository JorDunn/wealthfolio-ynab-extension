# wealthfolio-ynab-sync

YNAB transaction sync addon for Wealthfolio. Syncs transactions from a YNAB budget into mapped Wealthfolio cash accounts.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev:server

# Build for production
npm run build

# Run tests
npm test -- --run

# Package addon
npm run bundle
```
